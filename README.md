# 3 meses

Netflix-style memory site built with React, Vite, and TypeScript.

## Local preview

```bash
npm install
npm run dev
```

Open the URL shown by Vite. The app uses a GitHub Pages base path:

```text
/3-meses-carolina/
```

## Main pages

- `#/movie` - full movie
- `#/browse` - stages
- `#/episode/1` through `#/episode/5` - individual stages
- `#/credits` - credits

## Build

```bash
npm run build
```
